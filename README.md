# Sistema de Gestão para Assistência Técnica (Stop Cell)

## Sobre o projeto

Este projeto tem como objetivo o desenvolvimento de um sistema orientado a objetos para uma assistência técnica de celulares, com foco na organização dos atendimentos, controle de serviços e melhoria da comunicação com os clientes.

A proposta surgiu a partir da análise do funcionamento da empresa Stop Cell, onde foram identificadas dificuldades relacionadas ao uso de processos manuais, como registros em papel, falta de controle de histórico e falhas na comunicação.

---

## Objetivo

Desenvolver um sistema que permita automatizar a gestão de ordens de serviço (OS), tornando o processo mais organizado, eficiente e confiável.

---

## Funcionalidades

* Cadastro de clientes
* Cadastro de aparelhos
* Geração de ordens de serviço
* Controle de status dos serviços
* Histórico de atendimentos
* Registro de pagamentos (Pix e cartão)
* Envio de notificações para clientes

---

## Conceitos utilizados

O projeto foi desenvolvido com base nos principais conceitos de Programação Orientada a Objetos:

* Herança
* Polimorfismo
* Encapsulamento
* Classes abstratas
* Interfaces
* Composição e agregação

## Estrutura do sistema

As principais classes do sistema são:

* Cliente
* OrdemServico
* Aparelho
* Transacao (classe abstrata)
* TransacaoPix
* TransacaoCartao
* Notificacao

* ItemCheckList
